# 🦽 轮椅系列产品手册

轮椅是康复护理类目的核心大件。店长在接待时，请务必先弄清三个关键问题：**1. 谁推？（老人自推还是家属代推） 2. 在哪用？（居家还是常外出） 3. 楼层与出行工具？（有没有电梯、需不需要放后备箱）**，据此精准推荐型号。

---

## 一、可孚轻便折叠铝合金轮椅 【高频走量款】

::: tip 💡 核心一句话卖点
**航太级铝合金车架，单手可拎起！** 专为经常外出、需要上下楼或放汽车后备箱的家庭设计，免充气蜂窝减震胎，告别半路爆胎烦恼。
:::

### 📊 核心参数与规格
* **车架材质**：高强度铝合金（防锈防腐，重量通常仅为铁制轮椅的一半）。
* **折叠设计**：靠背可折叠，脚踏板可旋转或拆卸，折叠后体积极小。
* **轮胎类型**：PU 实心免充气避震胎（不用打气，扎不破）。
* **刹车系统**：前后四刹车设计（推行者和乘坐者均可控制刹车）。

### 💬 门店实战销售话术（化解材质差价）
::: info 🗣️ 顾客抗拒点化解（店长必背）
**🙋 顾客问：“你们这个铝合金的怎么比那种钢管的贵好几百？”**

**💁‍♀️ 标准话术答：** “叔叔/阿姨，钢管的虽然便宜，但是死沉死沉的！您想啊，推轮椅的往往也是上了年纪的老伴或者家属，遇到台阶或者要搬上汽车后备箱，钢管轮椅根本搬不动，还容易闪了腰。而且钢管时间长了容易生锈卡顿。咱们这款航太铝合金的，女生都能单手拎起来，不仅轻便结实，还永远不生锈，推着带老人出门遛弯轻松太多了，这几百块钱花在‘省力气’上绝对值！”
:::

## 二、可孚智能电动轮椅 【高客单利润款】

::: tip 💡 核心推荐场景
适合**上肢有活动能力、思维清晰但下肢行动不便**的老年人。彻底解放家属，老人可以自己推摇杆去公园、去买菜，重拾生活自主权。
:::

### 📊 核心参数与规格
* **操控方式**：360度智能摇杆（一根手指就能控制方向和速度，松手即停）。
* **刹车系统**：电磁刹车（遇到上下坡，松开摇杆立刻锁死不溜车，极其安全）。
* **电池选择**：可选铅酸电池（便宜但较重）或锂电池（轻便、续航长、可上飞机）。
* **双模切换**：支持电动/手动一键切换（没电了家属也能当普通轮椅推）。

## ⚠️ 售后与客诉排雷指南

轮椅尤其是电动轮椅，由于涉及机械结构和电池，如果在门店不教清楚，极易导致顾客拉回家就损坏！

::: warning ⛔ 轮椅交付与排雷三大纪律（提货必嘱咐）
1. **上下台阶怎么推？（手动轮椅必讲）** 👉 遇到台阶或小门槛，**千万不能硬推前轮撞过去！** 家属必须用脚踩住后面底部的“防倾翻踩踏杆”，双手下压把手，让前轮翘起来越过障碍物，硬撞会导致前小轮前叉断裂！
2. **为什么推不动？（电动轮椅必讲）** 👉 电动轮椅在“电动模式”下，轮子是锁死的，推不动是正常的！如果要推，必须把电机上的离合器扳到“手动模式”。
3. **电池饿死（电动轮椅必讲）** 👉 电池的头号杀手是“长期不用还不充电”。请反复叮嘱顾客：哪怕冬天一两个月推不出去，也**必须每个月给它充一次电**！一旦电池亏电饿死，厂家是不保修的，换一块电池要大几百块！
:::
