# 🛠️ 高频故障秒查 (店员现场自排查)

遇到设备报错，请直接对照下方代码进行初步排查。**全品类医疗器械 90% 的“故障”均为操作不当，按此步骤可现场解决，无需退换机！**

::: details 🌡️ 额温枪/体温计客诉：顾客反馈“数值高得离谱 / 屏幕报错 Lo / 测量不准”
🔍 故障原因：
机器坏的概率极低！99% 都是这四大超级乌龙：报错 Lo 误以为没电、单位变成了华氏度(℉)、模式选成了测物温、或者误调了温度补偿参数！
🛠️ 现场安抚与解决战术：
 * 安抚话术（把“不准”包装成“多功能高科技”）：
   > “姐，您先别急，真不是机器坏了。咱们可孚这款是多功能的医疗级机器，灵敏度极高。您在家可能是不小心按到了国外的计量单位，或者是刚从室外进来体表温度没回升。来，我帮您排查一下，一秒钟就能解决！”
   > 
 * 排查一：看报错，屏幕显示“Lo”！（最常见误解）
   * 现象：顾客气冲冲地说“你这机器刚买就没电了/坏了，一直显示字母 Lo！”
   * 实操解决：纠正误区！Lo 绝对不是没电（没电会显示电池图标），而是 Low Temperature（测到的表面温度低于人体正常的 32℃ 下限）。90% 是因为顾客刚从外面刮冷风进来额头冰凉，或者额头有冷汗，又或者距离太远直接测到了空气！让顾客在屋里暖和 3 分钟，擦干冷汗，探头靠近额头（1~3厘米）再测，瞬间出数！
 * 排查二：看单位，是不是变成了华氏度 ℉ ！（最吓人乌龙）
   * 现象：顾客惊恐地说“我测出来 97 度、98 度！这枪疯了！”
   * 实操解决：这是不小心把摄氏度（℃）切成了华氏度（℉）。长按“设置/Set”或相应的切换键，把 ℉ 调回 ℃。屏幕角落只要显示出 ℃，数值瞬间恢复正常的 36.X 度。
 * 排查三：看模式，是不是测物温了！（最高频乌龙）
   * 现象：测出来总是 32 度左右甚至更低。
   * 实操解决：检查屏幕是不是显示了**“小房子”或“奶瓶”图标（表面温度模式）。果断帮顾客一键切回“小人头”**图标（人体温度模式）。
 * 排查四：查参数，是不是调了补偿偏差！（最隐蔽乌龙）
   * 现象：不管怎么测，永远固定比正常体温高或低 1 度左右。
   * 实操解决：顾客误进了高级设置。长按设置键进入内部菜单（通常是 F3 或 F4 界面），看偏差参数是不是变成了 +1.0 等数字，立刻调回 0.0！（终极傻瓜化解法：直接扣掉电池，静置 1 分钟后重装，机器强制恢复出厂默认值）。
 * 排查五：清障碍，稳环境，准手法（物理操作规范）
   * 实操解决：额头有退热贴残留水分、厚粉底，都会导致折射偏低。必须擦干额头。探头必须垂直对准额头正中央（眉心上方），保持 1~3 厘米的距离。严禁死死贴在肉上，也严禁隔着半米远测空气。
:::

::: details 🫀 血压计客诉：屏幕显示 E-1~E-4 报错 / 顾客抱怨“跟医院测的不一样”
🔍 故障原因：
机器本身故障率不到 1%！99% 的 E 代码报错是因为**袖带没插紧、绑太松、测的时候说话乱动，或者衣服卷得太紧勒住了血管！**

🛠️ 现场安抚与解决战术：

* **安抚话术（把“报错”包装成“防误诊保护机制”）：**
  > “叔叔，您千万别急，机器绝对没坏！出现字母 E，恰恰说明咱们可孚的机器芯片太高级了。它察觉到您的袖带没绑好或者手动了一下，为了防止给您报一个错的血压，它强行拦截了！这叫防误诊保护，是对您的安全负责。来，我给您按标准手法绑一次。”

* **排查一：破除“医院迷信”（顾客说跟医院测的不一样）**
  * **现象**：顾客拿着机器来吵，说昨天在医院测 130，今天在家测 150，机器不准。
  * **实操解决**：这叫“白大褂效应”和“时间差”。**人的血压一天变三次！** 告诉顾客：“大爷，昨天下午在医院和今天早上在家，血压能一样吗？而且您拎着机器走几条街到店里，心跳加速，血压肯定高。您先坐着歇 5 分钟咱们再测！”

* **排查二：报错 E-1 / E-3（袖带气压问题，最高频）**
  * **现象**：充气很慢，或者充一下就泄气，屏幕显示 E-1 或 E-3。
  * **实操解决**：**漏气了！** 第一看袖带的管子有没有死死插到底座的孔里（很多老人只插了一半）；第二看袖带是不是绑得太松（必须能塞进两根手指，且袖带下边缘距离手肘关节 1~2 厘米）。

* **排查三：报错 E-2 / E-4（测量干扰问题）**
  * **现象**：充到一半突然停了报错。
  * **实操解决**：顾客测量时**说话了、咳嗽了，或者手乱动了**。最致命的是：顾客把厚毛衣死死卷到大臂上，像止血带一样勒住了血管（这叫人为截流，绝壁测不准）！**必须把厚衣服脱掉，隔着一件薄秋衣测都没问题。**

* **排查四：连续测，越测越低 / 越测越高**
  * **现象**：顾客不信邪，坐在那连续按开关测了 5 次，发现数值全不一样。
  * **实操解决**：绝对禁止连续疯狂测！血管被反复压迫，弹性还没恢复。**必须要求顾客：测完一次，解开袖带，等至少 2-3 分钟，再测第二次！** 取两次的平均值。

::: details 💨 制氧机客诉：机器像拖拉机 / 滴滴报警亮红灯 / 鼻吸管喷水
🔍 故障原因：
制氧机是大件，顾客买回去一有风吹草动就吓得要命。90% 的客诉是因为：**底部的绑带没剪断、刚开机氧浓度没上来、或者湿化杯的水装得太满了！**

🛠️ 现场安抚与解决战术：

* **安抚话术（建立医疗级设备的敬畏感）：**
  > “哥，这可是 5 升医用级制氧机，不是那种几十块钱的加湿器，它的安全报警系统极其灵敏！一有操作不当它就报警，这是为了保护里面的进口压缩机不被烧毁，也是为了确保给老爷子吸进去的都是高浓度纯氧。我教您几招，以后您就是半个制氧专家！”

* **排查一：机器震动极大，响声像拖拉机！（拆箱首发乌龙）**
  * **现象**：刚买回家，一开机整个机器在地上跳，嗡嗡巨响。
  * **实操解决**：**底部的运输固定扎带/螺丝根本没剪！** 为了防止快递摔坏压缩机，出厂时底部都有防震扎带。立刻告诉顾客：“赶紧关机！把机器放倒，把底部的扎带剪断抽出来，压缩机悬浮了就立刻静音了！”

* **排查二：开机滴滴狂响，屏幕亮红灯 / 黄灯（最容易吓退顾客）**
  * **现象**：按开机键后，机器一边嗡嗡响一边滴滴报警，氧气浓度显示只有五六十。
  * **实操解决**：这是**“低氧浓度报警”**，是完全正常的！制氧机里的分子筛刚开始工作，需要 3~5 分钟的“热身时间”才能把空气里的氮气排走，氧浓度才会爬升到 90% 以上。**告诉顾客：不要慌，让机器自己跑 3 分钟，绿灯一亮，报警自然解除！**

* **排查三：不出氧气，或者鼻管里往外喷水（湿化杯杀手）**
  * **现象**：机器正常转，但老人吸不到氧，或者吸了一鼻子水。
  * **实操解决**：这全是因为那个透明的“湿化杯”！
    1. **不出氧**：杯子盖子没拧紧，或者螺纹拧歪了，氧气全从缝隙漏光了！重新拧正拧紧。
    2. **喷水**：杯子里的水**加过了 MAX 最高水位线！** 氧气一翻滚，直接把水顶进了吸氧管里。把水倒掉一半，保持在两条线中间即可！

* **排查四：用了几个月突然报警，声音变闷**
  * **现象**：以前好好的，最近老是报警。
  * **实操解决**：**进气滤网堵死了！** 机器吸不到空气，差点被憋死。让顾客把机器背后的黑色海绵滤网拔下来，用水洗干净、彻底晾干再塞回去，机器瞬间复活。
:::
