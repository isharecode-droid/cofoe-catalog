# 🛠️ 高频故障秒查 (店员现场自排查)

遇到设备报错，请直接对照下方代码进行初步排查。**全品类医疗器械 90% 的“故障”均为操作不当，按此步骤可现场解决，无需退换机！**

::: details 🩸 血压计报错：屏幕显示 E-3 / E-4 / E-5 或气打不上来
**🔍 故障原因：** 袖带缠绕过松、接口漏气、或测量时说话乱动。
**🛠️ 现场安抚与解决战术：**
1. **安抚话术**：“叔叔，机器没坏，这是高灵敏度防误测保护启动了。您外套太厚压住管子了，我帮您脱了重测。”
2. **实操**：重新绑紧袖带（标准：绑好后刚好能插进一根手指）。
3. **检查接口**：拔下连接血压计的导气管插头，**用力重新插到底**。
4. 重启设备，要求顾客保持绝对安静，重新测量。
:::

::: details 💨 制氧机报错：亮红灯并伴随“滴滴”持续警报声
**🔍 故障原因：** 常见为湿化杯未拧紧漏气，或机器背部滤芯严重堵死。
**🛠️ 现场安抚与解决战术：**
1. **安抚话术**：“阿姨别急，这是制氧机在提醒您该做保养了，就像汽车要换机油一样，我给您检查一下。”
2. **查水杯**：检查湿化杯（水杯）是否拧歪？里面的橡胶垫圈还在不在？**重新对正螺纹死死拧紧**。
3. **查滤网**：抠开背部进气口，如果里面的海绵发黑堵死，立刻换一张新的过滤棉（如果没有，暂时不放海绵测试是否还报警）。
4. 若以上无效，说明内部电磁阀故障，请引导顾客走售后流程。
:::

::: details 🌡️ 额温枪/体温计客诉：顾客反馈“数值高得离谱 / 屏幕报错 Lo / 测量不准”
🔍 故障原因：
机器坏的概率极低！99% 都是这四大超级乌龙：报错 Lo 误以为没电、单位变成了华氏度(℉)、模式选成了测物温、或者误调了温度补偿参数！
🛠️ 现场安抚与解决战术：
 * 安抚话术（把“不准”包装成“多功能高科技”）：
   > “姐，您先别急，真不是机器坏了。咱们可孚这款是多功能的医疗级机器，灵敏度极高。您在家可能是不小心按到了国外的计量单位，或者是刚从室外进来体表温度没回升。来，我帮您排查一下，一秒钟就能解决！”
   > 
 * 排查一：看报错，屏幕显示“Lo”！（最常见误解）
   * 现象：顾客气冲冲地说“你这机器刚买就没电了/坏了，一直显示字母 Lo！”
   * 实操解决：纠正误区！Lo 绝对不是没电（没电会显示电池图标），而是 Low Temperature（测到的表面温度低于人体正常的 32℃ 下限）。90% 是因为顾客刚从外面刮冷风进来额头冰凉，或者额头有冷汗，又或者距离太远直接测到了空气！让顾客在屋里暖和 3 分钟，擦干冷汗，探头靠近额头（1~3厘米）再测，瞬间出数！
 * 排查二：看单位，是不是变成了华氏度 ℉ ！（最吓人乌龙）
   * 现象：顾客惊恐地说“我测出来 97 度、98 度！这枪疯了！”
   * 实操解决：这是不小心把摄氏度（℃）切成了华氏度（℉）。长按“设置/Set”或相应的切换键，把 ℉ 调回 ℃。屏幕角落只要显示出 ℃，数值瞬间恢复正常的 36.X 度。
 * 排查三：看模式，是不是测物温了！（最高频乌龙）
   * 现象：测出来总是 32 度左右甚至更低。
   * 实操解决：检查屏幕是不是显示了**“小房子”或“奶瓶”图标（表面温度模式）。果断帮顾客一键切回“小人头”**图标（人体温度模式）。
 * 排查四：查参数，是不是调了补偿偏差！（最隐蔽乌龙）
   * 现象：不管怎么测，永远固定比正常体温高或低 1 度左右。
   * 实操解决：顾客误进了高级设置。长按设置键进入内部菜单（通常是 F3 或 F4 界面），看偏差参数是不是变成了 +1.0 等数字，立刻调回 0.0！（终极傻瓜化解法：直接扣掉电池，静置 1 分钟后重装，机器强制恢复出厂默认值）。
 * 排查五：清障碍，稳环境，准手法（物理操作规范）
   * 实操解决：额头有退热贴残留水分、厚粉底，都会导致折射偏低。必须擦干额头。探头必须垂直对准额头正中央（眉心上方），保持 1~3 厘米的距离。严禁死死贴在肉上，也严禁隔着半米远测空气。
     :::
