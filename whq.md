# 🌬️ 雾化器系列产品手册

雾化器是治疗呼吸道疾病（如哮喘、急慢性支气管炎、小儿咳嗽等）的核心给药设备。**秋冬换季是雾化器的绝对销售旺季**。店长在推荐时，请务必根据顾客是**居家长期用**还是**追求静音便携**来推荐合适的款式。

---

## 一、可孚医用压缩式雾化器 <Badge type="tip" text="家庭/儿童首选" />

::: tip 💡 核心一句话卖点
**出雾细腻直达肺泡，医院同款压缩技术！** 纯铜无油压缩机，动力强劲寿命长，雾化颗粒细腻均匀，孩子吸着不呛嗓，吸收效果极佳。
:::

### 📊 核心参数与规格
* **雾化颗粒**：中位粒径 ≤5μm（能深入下呼吸道和肺泡，药效吸收更好）。
* **核心部件**：医用级无油压缩泵，稳定耐用，可持续长时间工作。
* **噪音控制**：升级减震降噪技术，运行声音平稳不刺耳。
* **出雾调节**：可通过调节拨片控制出雾量大小，大人小孩都适用。

### 💬 门店实战销售话术（打消宝妈顾虑）
::: info 🗣️ 顾客抗拒点化解（店长必背）
**🙋 顾客问：“买个机器在家里做雾化，效果能有医院的好吗？”**

**💁‍♀️ 标准话术答：** “宝妈您放心，咱们这款用的是和医院一样原理的‘空气压缩技术’，出来的雾化颗粒小于 5 微米，能直接吸进气管和肺部。在家里做，一方面避免了去医院排队交叉感染，另一方面孩子在熟悉的环境里不害怕、不哭闹，配合度高了，吸收效果反而比在医院一边哭一边做要好得多！而且机器买回去，全家人都能用，以后感冒咳嗽用生理盐水雾化润润嗓子也是极好的。”
:::

## 二、可孚手持微网式雾化器 <Badge type="info" text="便携/差旅必备" />

::: tip 💡 核心推荐场景
适合**频繁出差的成年人**，或者**对声音极度敏感、只能等睡着了才肯做雾化的婴儿**。真正做到“口袋里的雾化器，零噪音运行”。
:::

### 📊 核心参数与规格
* **体积重量**：极其轻巧，单手握持，外出可轻松装进包里。
* **噪音表现**：近乎零噪音（超声震动微网出雾），夜晚宝宝熟睡时也能悄悄做。
* **供电方式**：干电池 / USB 插线双模供电，随时随地可用。
* **残液量**：杯底倾斜设计，药液残留极少，不浪费昂贵药水。

## ⚠️ 售后与客诉排雷指南

雾化器 80% 的售后问题（如不出雾、出雾变小）都是因为**清理不当导致微孔堵塞**！门店交付时，必须严厉嘱咐清洗保养规范：

::: warning ⛔ 雾化器清洗与保养红线（提货必嘱咐）
1. **用完必须立刻清洗！** 👉 药液非常容易结晶，每次做完雾化后，**必须立刻**用温水清洗药杯，并加入少量温水再开机空喷几十秒，把喷孔里的残药冲洗干净。否则第二天微孔就会被死死堵住，导致机器报废！
2. **微网雾化器测机禁忌** 👉 顾客买微网款（便携款）时，如果要在店里试机，**绝对不能用纯净水或蒸馏水试！** 微网技术需要导电性，纯净水不导电会导致不出雾，请务必用**生理盐水或自来水**试机。
3. **配件是消耗品** 👉 提醒顾客，面罩、咬嘴和药杯属于个人卫生消耗品，为了防止细菌滋生，建议每 3 个月左右更换一次（这也是门店锁客复购的绝佳话术）。
:::
