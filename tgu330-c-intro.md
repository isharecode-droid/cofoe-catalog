## 📦 核心物理参数与优势翻译（包装盒密码）

不要让顾客自己看冰冷的参数，请店长直接用以下话术进行“降维翻译”：

::: tip 🥇 1. 纯银电极 (核心精准壁垒)
* **硬件级优势**：弃用低端碳电极，采用纯银电极导电，抗干扰极强，生化反应传导更稳定。
* **顾客利益点**：“阿姨，这台机器用的是纯银电极，银的导电性最好，测得极快、数据极其稳定。这就是医疗级精准度的保障！”
:::

::: danger 🔄 2. 2合1 双重测试 (高性价比之选)
* **硬件级优势**：一台仪器，一条试纸，同时检测两项指标。
* **顾客利益点**：“扎一次针，出一滴血，血糖和尿酸同时出结果。少花买另一台机器的钱，最关键是少受一次扎针的罪！”
:::

::: info 🗣️ 3. 语音播报 & 💡 背光大屏 (极致适老化)
* **硬件级优势**：全程大音量语音指导及结果播报，搭配高清夜视背光屏幕。
* **顾客利益点**：“测完不用到处找老花镜，机器像小护士一样把数值大声报给您听！半夜起夜测，屏幕带背光，黑灯瞎火也看得清清楚楚。”
:::

::: warning 🪪 4. 专属调码卡校准 (批次零误差)
* **硬件级优势**：每盒试纸均配有专属“调码卡”，机器读取卡片代码以匹配当前批次生物酶活性，拒绝一刀切。
* **顾客利益点**：“每批试纸都有专属芯片校准，保证您测出来的数值跟医院抽血一样准，多插一张卡，绝不拿您的健康开玩笑！”
:::

## 🎯 目标人群精准拦截 (什么时候推它？)

遇到以下三类顾客，**必须强推 TGU300-C 进行药械关联**：
1. **拿降糖药的顾客**（二甲双胍、胰岛素）：高血糖极易引发肾脏代谢异常，导致尿酸升高，必须双测。
2. **拿痛风药/贴膏药的顾客**（非布司他、秋水仙碱）：骨关节疼只是表象，尿酸指标偏高才是根本。
3. **体型偏胖/经常应酬饮酒的顾客**：海鲜配啤酒，属于代谢综合征绝对高危人群。
